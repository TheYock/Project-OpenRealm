Chat Effects

Type an effect prefix before your message to animate it in your bubble:

wave:Characters bob in a sine wave
wave2:Steeper wave, alternating red/yellow
shake:Characters jitter randomly
scroll:Text slides side to side
glow1:Red → yellow cycling gradient
glow2:Blue → cyan cycling gradient
glow3:Green → lime cycling gradient
flash1:Flashes red / yellow
flash2:Flashes blue / cyan
flash3:Flashes green / white

Chat Colors

Add a colour prefix before your message or effect:

red: green: blue: yellow: cyan: purple: white: orange: pink: lime:

Combine colour + effect: red:wave:Hello!

Colour only: blue:Hello!
